# MindCare AI

MindCare AI is a Web-Based Intelligent Mental Support System designed to help students manage stress, anxiety, and emotional challenges through AI-assisted conversations.

## Features

- AI Therapist Chat
- Emotion Detection using Transformer NLP
- Mood Tracking System
- Personal Journal
- Psychological Distress Detection
- Suicide Risk Detection
- Government Crisis Helpline Integration

## Technologies

- Python
- Streamlit
- NLP (Natural Language Processing)
- Transformer Models
- AI Emotion Detection
- Computational Psychology
